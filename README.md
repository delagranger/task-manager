# ABOUT
Just an easy console task-manager on Python, designed with OOP

## USING
### 1. Create venv
 - type to initialize venv, choose console_type from venv\Scripts:
    - python -m venv venv
    - venv\Scripts\\(console_type)
 - type to install requirements:
    - pip install -r requirements.txt
### 2. Create .env for logging
 - create .env file in project`s root folder
 - put LOG_LEVEL=(logging level) in .env (choose logging level from DEBUG, INFO, WARNING, ERROR or CRITICAL)
